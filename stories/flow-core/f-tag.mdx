# f-tag

<br />

## Overview

<f-divider />

Tags allow users to categorize the content. They can be used to add metadata to an element such as category, or property or show a status.

## state

<f-divider />

States on Tags are used to communicate purpose and it’s connotation. For example, a red color connotes danger,
whereas a green color connotes success and so on.

<br />

<p className="note-section">
	Note: To use a tag of different color other than the state colors, use fill property.
</p>

## counter

<f-divider />

Counter property enables a counter on the tag.

The counter component by default abbreviates the value (1000 = 1k). You can learn more about the counter
[here](/docs/components-f-counter--anatomy).

<br />

<p className="note-section">
	Note: Counter is always on the right end of the button, except if icon-right=”i-close”. In that
	case counter comes on the left of the close icon.
</p>
